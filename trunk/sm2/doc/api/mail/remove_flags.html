remove_flags

boolean remove_flags (int $message_num, array $flags[])

$flags is an array of flags to be removed. If those flags are not set, don't report any errors, just remove the flags that ARE set and are in the $flags array. This allows for clearing of all flags by sending an array with all flags in it. The flags can be: recent, unseen, answered, deleted, draft, or flagged.

Returns:

True if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
