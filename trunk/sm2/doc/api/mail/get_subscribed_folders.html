get_subscribed_folders

array get_subscribed_folders (string $pattern)

Returns an array of objects. Each object contains information about a single folder. This should list only folders that the user is subscribed to and that match the $pattern parameter. These objects have the following properties:
   name - the name of this mailbox
   full_name - the entire name of this mailbox, including the path
   path - the path leading up to this folder
   delimiter - the mailbox delimiter (usually '/')
   attributes[] - array of attributes (noinferiors, noselect, marked, or unmarked)


Returns:

That object if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
