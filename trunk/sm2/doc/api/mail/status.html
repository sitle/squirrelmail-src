status

object status ([$folder])

Returns information about the current folder in an object. The following are properties of that object:
  • name - name of the folder
  • num_msgs - number of messages
  • num_recent - number of recent messages
  • num_unread - number of unread messages

In case of errors, set the $error variable and return false.

If no $folder is specified, then assume $this->folder. However, with this, it is a requirement that $this->select($folder) has been called previous to this. If this hasn't happened, in this case consider it an error and return a new instance of the Error class (also set the $this->error variable).
