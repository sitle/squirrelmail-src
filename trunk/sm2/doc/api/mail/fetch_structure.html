fetch_structure

object fetch_structure (int $message_id)

Returns an object with the decoded body structure of the specified message. The object has the following properties:
  • type - the primary body type
  • subtype - MIME subtype
  • encoding - body transfer encoding
  • description - the description
  • id - identification string
  • num_lines - number of lines
  • num_bytes - number of bytes
  • disposition - disposition string
  • parameters [a] - array of parameters
  • parts [b] - array of objects for each message part
[a] - This is an array of objects that have an "attribute" and "value" property each.
[b] - This is an array of objects identical to this one that will describe the entire message. With the exception that they cannot have the "parts" property. This will be a flat list of all parts if the message is multipart.

Returns:

That object if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
