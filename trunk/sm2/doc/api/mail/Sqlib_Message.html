sqapi_mail -- constructor

boolean sqapi_mail (string $username, string $password, string $mail_server, int $mail_port, string $folder)

As the constructor of the class, this function will create a connection to themail server, logging them in, and selecting the specified folder ($folder). All variables passed in should be set as global variables to the class ($_username, $_password, $_host, $_port, $_folder, $_stream). The $_stream variable is the stream that was created.

Returns:

True if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
