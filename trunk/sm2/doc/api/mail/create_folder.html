create_folder

boolean create_folder (string $folder_name)

Creates a folder with the given namei. Note that folders with international characters should be encoded with the PHP function imap_utf7_encode(), or something equivilent. UTF-7 encoding is defined in RFC 2060, section 5.1.3.

This function should automatically subscribe them to the folder as well.

Returns:

True if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
