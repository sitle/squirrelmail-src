rename_folder

boolean rename_folder (string $current_folder, string $new_folder)

Renames a specified folder to the new folder name. This can also be used for moving folders. See the note on create_folder() for info on how to encode the folder name for international characters.

This function should unsubscribe from $current_folder and subscribe them to $new_folder, and return false if there are any errors at all.

Returns:

True if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
