fetch_body

string fetch_body (int $message_id, string $part_number)

This fetches a particular part of the message body (specified by $part_number). The $part_number is a period-delimited list of numbers as specified in IMAP4 specification (2.4.1, etc). Body parts are not decoded here.

Returns:

A string with the body of the message if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
