fetch_brief_headers

array fetch_brief_headers ([int $start, [int $end]])

If no parameters are specified, $start = 1, and $end = total messages in the folder (list all messages).
If only $start is specified, $end will equal the total number of messages in the folder.
Returns a summary of all the messages in the current mailbox. It will return an array of objects, one array element for each message. Each array element will have an object with the following properties: 
      id
      subject
      date
      from
      size
      flags[] (a)
(a) An array of flags in lower case ("deleted", "answered", etc).

Returns:

That array if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned.
