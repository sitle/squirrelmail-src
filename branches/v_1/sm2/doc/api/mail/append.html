append

boolean append (string $message, string $folder, array $flags)

Appends a message to the specified folder. Often this is used to save sent messages or something to that degree. The $message string must be in the form of a standard text message with standard headers and body. It can be multipart or anything your heart desires. Lines must end with \r\n to be RFC complient!!.

The flags array can contain any of the following elements (one per element): recent, unseen, answered, deleted, draft, or flagged.

Returns:

True if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
