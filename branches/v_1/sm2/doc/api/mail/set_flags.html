set_flags

boolean set_flags (int $message_num, array $flags[])

Sets flags on the specified message_num. The flags are an array of strings that can have the folowing elements: recent, unseen, answered, deleted, draft, or flagged.

Returns:

True if successful, false if failure and sets the $error variable with a reasonable error message and the $this->error variable will be set.
