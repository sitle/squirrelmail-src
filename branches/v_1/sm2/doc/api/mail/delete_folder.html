delete_folder

boolean delete_folder (string $folder_name)

Deletes a foder from the mail server. This function should automatically unsubscribe them from the folder as well.

Returns:

True if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
