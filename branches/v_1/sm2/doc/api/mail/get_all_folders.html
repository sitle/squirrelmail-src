get_all_folders

array get_all_folders(string $pattern)

Exactly the same as get_subscribed_folders() except this lists all folder matching the $pattern parameter (not just subscribed folders).

Returns:

That object if successful. If unsuccessful, an instance of the Error class with the appropriate error number set will be returned and the $this->error variable will be set.
